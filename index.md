# Super Mario Eclipse Wiki
Since Super Mario Eclipse is so new this Wiki is a MASSIVE work in progress. Regardless this wiki is a great starting  
point for learning about SME speedrunning, access to multiple resources, and general information to benefit the community.  
Here is a link to the [Eclipse Speedruns Discord](https://discord.gg/3Zx3QXAcXu).  
Here is a link to the general [Mario Eclipse Discord](https://discord.gg/zfKbppaZM7) for more casual play or more general questions.
